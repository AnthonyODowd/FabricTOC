# <a name="PeerCommand"></a> Peer Command

The `peer` command allow administrators to interact with a peer.

## Syntax

The `peer` command has 5 different subcommands within it as follows:

```
peer chaincode   
peer channel     
peer logging     
peer node        
peer version     
```
